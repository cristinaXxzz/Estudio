📖 Estudio 对话式复习
科目
    🌵

    左边选一个科目，或者点「＋ 新科目」建一个。

    建好科目 → 上传资料 → AI 拆成知识点 → 开始边聊边复习。

    资料

    支持 txt / md / pdf / docx / pptx / xlsx / csv / html / 图片。手写笔记（如 GoodNotes 导出的 PDF）和拍照图片会让 AI 帮你认字——需要你的模型支持看图。上传后 AI 自动拆成知识点。

      知识点

        它记住的你

        每次复习后 AI 自动提炼的学习记忆（薄弱点、易混淆、对你管用的讲法…），下次复习会自动带上。不对的可以删。

          复习记录

            设置

            AI 接口 1（主力，聊天用）

            AI 接口 2（可选：评分裁判 + 备胎，拆资料时俩接口一起干活）

            记忆向量接口（可选，给"学习记忆"提速）

            填一个 Embedding（向量）接口后，AI 的记忆检索会更聪明——按"意思相近"找记忆，而不只是字面匹配。不填也能用（退化为关键词检索）。硅基流动等服务有免费的向量模型。

            GitHub 云备份

            填一个 GitHub Token（钥匙），数据会自动备份到你账号下的一个私有仓库。怎么申请 Token 见 README。

            🎭 AI 角色

            这些是能拉进群聊陪你复习的 AI 同学。可以改人设、换接口，也可以自己造新的。

              编辑角色

              👥 谁来陪你复习？

              勾选要拉进群的 AI 同学（1 个就是单聊，2~3 个群聊最热闹）。想加新角色去「🎭 角色」。
